Ejemplo 1:

1_ La propiedad de CSS que se utiliza para convertir un contenedor en un contenedor flex, seria "Display".

2_ La propiedad de CSS que se utiliza para centrar el contenedor, seria "justify-content: center".

3_ La propiedad de XSS que se utiliza para centrar el contenedor a lo largo del eje vertical, seria "align-items: center".

4_ La altura del contenedor especificada en el codigo es de 200px.

5_ El ancho de la caja en el codigo es de 100px.

6_ El color establecido en la caja es el rojo.

7_ La etiqueta utilizada en HTML para crear el contenedor es la etiqueta "div".

8_ Al contenedor se le aplica la clase "container".

9_ A las cajas se le aplica la clase "box".

10_ El proposito principal del codigo CSS, es el de mostrar un cuadrado rojo.

Ejemplo 2:

1_ La propiedad de CSS se utiliza para convertir un contenedor en un contenedor flex, seria la propiedad "Display".

2_ El valor que se utiliza en la propiedad "flex-wrap" seria el valor "wrap".

3_ La propiead que se utiliza para centrar los elementos por el eje principal horizontal seria la propiedad "justify-content".

4_ La propiedad que se utiliza para centrar los elemenetos a lo largo del eje secundario vertical, seria la propiedad "align-items".

5_ El proposito del valor "justify-content: space-between", seria el de distribuir los elementos a lo largo del eje principal horizontal.

6_ El proposito del valor "Baseline" seria el de alinear los elementos a lo largo de la linea de la base.

7_ La propiedad de CSS que se utiliza para establecer el ancho del contenedor del titulo header, seria la propiedad "width".

8_ La propiedad de CSS que se utiliza para alinear el texto, seria la propiedad "text-align".

9_ La propiedad de CSS que se utiliza para establecer el margen, seria la propiedad "margin".

10_ La propiedad de CSS que se utiliza para establecer el color de fondo del contenedor principal seria la propiedad "background-color".
